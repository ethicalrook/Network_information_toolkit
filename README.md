# Project Title:
Network Information Toolkit

# Project Description
I built this project for getting the basic network information ( IP address, IP public or private, Registrar name etc.)
<br>
It takes the input of the domain and gives the basic information about that domain.
<br>
This toolkit is for the students who wants to learn basic networking and learn how the networks works

# Features

Features includes to retrive ip address ip type by domain ( version 1).

# Technologies used

1. python
2. Python libraries
3. tldxtract ( checks if the entered domain is valid or not)
4. whois ( checks and finds the basic information like registrar etc.)
5. socket ( Takes the domain by input and gives the ip address of it )
6. ipaddress ( Checks if the ip address is public or private )
